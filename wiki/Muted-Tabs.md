# Muted Tabs

## Chrome

By default, Chrome does not allow users to unmute a tab if it was muted by an extension (Twitch Lurker). You have to change a setting (Chrome flag) to fix this. Chrome only lets you mute/unmute a "site". Most other browsers give the user this basic feature, but Chrome requires you to do the following if you want to unmute the tab:

1. Navigate to `chrome://flags/#enable-tab-audio-muting` in the address bar.
1. Set `Tab audio muting UI control` to `Enabled`.
1. Close and relaunch Chrome for the updated setting to apply.
1. Now, you can click the "muted sound" icon in the tab and it will toggle between muted/unmuted.
